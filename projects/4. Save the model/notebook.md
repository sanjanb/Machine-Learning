## SAVE THE MODEL USING PICKLE


```python
import pickle
```


```python
# save the python object

with open('model_pickle', 'wb') as f:
    pickle.dump(reg, f)
```


```python
with open('model_pickle', 'rb') as f:
    model = pickle.load(f)
```
